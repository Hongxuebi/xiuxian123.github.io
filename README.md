# 爱助手 - 鸿蒙原生私人AI助理 V6.3.1

基于鸿蒙 WebView 套壳 + HTML5/JS 前端，构建**多智能体架构的私人 AI 助理**。  
每个智能体拥有独立的系统提示词、用户画像、记忆库和对话历史，支持分批次记忆更新、双模式检索（预检索+函数调用），数据全量明文存储在鸿蒙公共目录，卸载不丢失。

**定位**：不只是工具，而是**陪伴型AI搭档**——懂你、记住你、主动关心你。

---

## ✨ 核心特性

### 🧠 智能体与记忆
- **多智能体**：可创建"医疗顾问""代码助手""学习教练"等多个智能体，各智能体数据完全隔离，切换自如。
- **多会话对话**：每个智能体内支持多个会话，话题互不干扰，共享该智能体的记忆库。
- **结构化记忆库**：记忆按分类文件存储（如 `工作发展.md`），AI 可自动扩展分类；支持增量更新、冲突检测与人工确认。
- **智能检索**：双模式检索（关键词预检索 + 函数调用检索），按智能体独立索引，毫秒级响应。
- **分批次记忆更新**：超长内容自动拆分批次，批次间传递摘要，确保模型不"失忆"，更新前自动备份。

### 📝 备忘录系统（V5.5~V6.0 持续升级）
- **IndexedDB持久化**：备忘录从纯内存存储升级为IndexedDB，关闭浏览器数据不丢失
- **AI联动**：AI可通过工具直接操作备忘录（搜索、创建、更新、删除、批量操作）
- **情绪日记**：AI感知用户情绪变化，自动记录到"情绪日记"文件夹，标题包含情绪类型
- **标签建议**：AI创建备忘录时自动建议标签（最多3个，每个2-4字）
- **文件夹树**：支持多层嵌套文件夹，右键菜单可新建子文件夹、移动、重命名、删除
- **AI整理**：一键让AI帮你提炼要点、建议标签、重新组织内容
- **筛选角标**：全部/收藏/已删除按钮实时显示数量
- **回收站**：最近删除（软删除+永久删除），支持多选恢复或清空
- **多选批量操作**：多选模式支持批量收藏/删除/恢复/移动/永久删除

### 🔍 联网搜索（V6.0 新增）
- **百度千帆API接入**：支持实时联网搜索，获取最新信息
- **AI自动判断**：联网开关开启时，AI根据问题类型自动决定是否联网

### 🎭 情感感知（V5.6 新增）
- 感知6种情绪：开心、平静、焦虑、难过、烦躁、疲惫
- 用户情绪低落时：先回应感受，再解决问题；不说教、不灌鸡汤
- 自动记录情绪日记到备忘录，后续可查询"最近心情怎么样"

### 🎨 用户体验
- **富文本编辑**：contenteditable 编辑器，支持待办事项、字号调整、颜色标记、图片插入（开发中）
- **待办事项**：圆形checkbox，回车自动新建，退格智能删除，双击checkbox删除整行
- **主题切换**：支持4个主题（默认、现代简约、东方美学、活力多彩），实时预览，自动保存。
- **卡片左滑菜单**：支持左滑露出操作按钮（收藏、置顶、移动、删除），回弹动画流畅。
- **深度思索（DeepSeek Reasoner）**：支持深度思索模式，AI思考过程可见

### 💾 数据自主可控
- 所有数据明文存储于 `Download/私人AI助理数据/`，手机文件管理器可直接查看、编辑
- 应用卸载数据不丢失
- 支持手动全量备份、周期自动备份

### 🚀 纯前端实现
- 核心功能 HTML5 + 原生 JS，模块化拆分（34个JS模块）
- 鸿蒙原生层仅处理权限、文件桥接和通知，无需学习 ArkTS
- 可在浏览器直接调试，开发效率高

| 模块 | 选型 |

## 🚀 快速开始（开发调试）

1. **克隆代码**（前端部分）  
   将本仓库前端代码下载到本地。

2. **在浏览器中测试**  
   直接用 Chrome 打开 `index.html`，所有功能均可离线运行（对话和记忆更新需配置 API 密钥）。

3. **配置 API 密钥**  
   在应用设置页面填入 DeepSeek API 密钥和百度搜索 API 密钥（可选，联网搜索用）。

4. **鸿蒙打包**  
   - 用 DevEco Studio 创建鸿蒙工程，将前端文件复制到 `entry/src/main/resources/rawfile/`。
   - 在 `entry/src/main/ets/pages/Index.ets` 中创建 WebView，加载 `rawfile://index.html`。
   - 配置 `module.json5` 和 `config.json` 申请所需权限（网络、存储、通知等）。
   - 编译生成 `.hap` 安装包，安装到鸿蒙设备。

## 📖 使用说明（核心流程）

### 1. 智能体管理
- 点击左上角菜单图标 → 选择"切换智能体"查看已有智能体。
- 点击"新建智能体"填写名称、系统提示词、用户画像，自动生成文件夹。

### 2. 对话
- 在对话界面选择当前智能体，输入问题，AI 将基于该智能体的角色和记忆回答。
- 支持无痕会话（不保存历史、不写入记忆）。
- **联网搜索**：开启联网开关，AI可实时搜索网络获取最新信息。
- **深度思索**：开启深度思索，AI使用DeepSeek Reasoner模型进行更深入的分析。

### 3. 记忆更新
- 手动点击"一键更新记忆"按钮。
- 系统自动收集新增对话和备忘录内容，若超过阈值自动分批次发送给 AI。
- 更新完成后自动重建索引。

### 4. 备忘录管理（V5.5+）
- 切换到备忘录面板，点击卡片查看详情或编辑
- 文件夹分类，支持嵌套子文件夹，右键可新建/移动/重命名/删除文件夹
- **AI可直接操作**：说"帮我记一下..."AI会调用create_memo工具
- 情绪日记：AI感知到情绪时会自动记录，你问"最近心情怎么样"AI能查询
- **AI整理**：在备忘录面板点击"🤖 整理"按钮，AI自动提炼要点、建议标签
- **回收站**：最近删除的备忘录可在回收站恢复或永久删除

### 5. 批量操作（V6.0）
- 点击右上角"多选"按钮进入多选模式
- 点击卡片选中，支持全选/反选
- 底部操作栏提供批量收藏/删除/移动/永久删除
